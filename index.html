How to define a table? Describe when to use the following HTML elements: <tr>, <th>, <td>, <thead>, <tfoot>, <caption>.
- <table> .... </table>
- <tr>: the table row.
- <th>: the heading cell in the table.
- <td>: the data cell in the table.
- <thead>: group header content in the table. It should have one or more <tr> inside.
- <tfoot>: group footer content in the table. It should have one or more <tr> inside.
- <caption>: define a title for the table. By default, it will be center-aligned above the table.

What is the difference between colspan and rowspan?
- colspan allows a single table cell to span the width of more than one cell or column, while rowspan is to allow a single table cell to span the height of more than one cell or row.

What is the difference between cells spacing and padding?
- cells spacing refers to the distance between the cells in a grid layout, while cell padding refers to the space around each cell.

How do you define table borders using CSS?
- <table border="2"> .... </table> (2 means 2 pixels)

How do you control border widths using CSS?
- border-width: top_and_bottom_val left_and_right_val;
